Installation
============

From PyPI
---------

.. code-block:: bash

   pip install reverse-abliterate

From Source
-----------

.. code-block:: bash

   git clone https://github.com/Carlos-Projects/reverse-abliterate.git
   cd reverse-abliterate
   pip install -e .

Dependencies
------------

- Python >= 3.11
- typer >= 0.12
- rich >= 13

Optional:

- torch (for forward hook detection)
